Задание

Разработайте функцию void PrintTree(ostream& dst, const path& p), которая принимает на вход путь к папке и выводит в поток, задаваемый параметром dst, дерево файлов и каталогов этой папки. Элементы папки следует вывести в обратном алфавитном порядке.

Формат выходных данных

Первым выведите имя папки, переданной на вход.

Напечатайте на одной строчке название одного объекта файловой системы — файла или папки. Печатайте только последнюю часть названия, получаемую из пути методом filename. Пути перед печатью конвертируйте в строку методом string.

Для вложенных элементов добавляйте отступ — два пробела.

Ограничения

Используйте эффективный контейнер и эффективную сортировку. Учтите, что статус файла, из которого можно понять, является ли объект файлом или папкой, уже хранится в итераторе директории. Не стоит вызывать status повторно.

Пример

Пусть дана следующая структура файлов и каталогов:

  • test_dir
    • a
      • f3.txt
    • b
      • f1.txt
      • f2.txt

В функцию передан путь test_dir. В этом случае в поток должен быть выведен следующий текст:

test_dir
  b
    f2.txt
    f1.txt
  a
    f3.txt

Этот пример можно найти в заготовке кода.

Что отправлять на проверку

Пришлите файл main.cpp, содержащий функцию void PrintTree(ostream& dst, const path& p).

Как будет тестироваться ваш код

Мы проверим, что:

  • Программа правильно составляет дерево файлов,
  • Выводит их в нужном порядке и формате,
  • Делает это эффективно.

Подсказка

  • Разработайте рекурсивную версию PrintTree со следующей сигнатурой:

    void PrintTree(ostream& dst, const path& p, const filesystem::file_status& status, int offset)

    Если status говорит о том, что p — папка, она проходит по всем объектам этой папки и рекурсивно вызывает себя.

  • Сложите все элементы в вектор и отсортируйте его алгоритмом sort. При этом следует хранить как путь, так и статус, значит, придётся использовать свой компаратор для сортировки.

  • Нужное количество пробелов можно получить таким образом: string(offset, ' ').

  • Имя файла из пути можно получить так: p.filename().string().

  • Каждый вызов рекурсивной функции PrintTree делает только один вывод в поток.
