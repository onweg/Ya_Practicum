Задание

Код для работы с деревом двоичного поиска, который использует сырые указатели, неустойчив к исключениям. Примените умный указатель unique_ptr для владения узлами дерева, чтобы сделать код надёжным: в структуре TreeNode замените сырые указатели на дочерние узлы на unique_ptr. Для хранения адреса родительского узла следует оставить обычный указатель, ведь родительский узел владеет дочерними узлами, а не наоборот.

Реализуйте функции для работы с деревом.

  • N — создаёт узел дерева, который содержит значение типа int и ссылается на два дочерних узла. Возвращает уникальный указатель на созданный узел. Переданные дочерние узлы должны ссылаться на созданный узел как на своего родителя. Дочерние узлы не должны иметь родителей. В противном случае функция должна выбросить исключение invalid_argument.
  • CheckTreeProperty — проверяет, является ли дерево двоичным деревом поиска, а именно:
    • оба поддерева являются двоичными деревьями поиска;
    • у всех узлов левого поддерева произвольного узла X значения ключей меньше либо равны значению ключа узла X;
    • у всех узлов правого поддерева произвольного узла X значения ключей больше либо равны значению ключа узла X.
  • begin — возвращает указатель на узел двоичного дерева поиска с наименьшим значением ключа. Гарантируется, что в эту функцию передаётся ненулевой указатель на узел двоичного дерева поиска.
  • next — возвращает указатель на следующий по величине узел двоичного дерева поиска либо nullptr, если текущий узел — последний. Гарантируется, что в эту функцию передаётся ненулевой указатель на узел двоичного дерева поиска.

Ограничения

Реализация должна быть эффективной и проходить ровно столько узлов дерева, сколько требуется пройти до нужной позиции. Сложность функции next должна быть амортизированной \(O(1)\).

Содержимое узлов учитывать не надо.

Что отправлять на проверку

Отправляемый на проверку код должен содержать все функции и структуры, которые были даны в заготовке решения. Функция main при её наличии будет заменена на версию из тренажёра.

Тренажёр проверит работу функций CheckTreeProperty, begin, next и N.
