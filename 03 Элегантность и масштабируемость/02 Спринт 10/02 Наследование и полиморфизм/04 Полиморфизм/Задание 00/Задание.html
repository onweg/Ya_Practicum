Задание

Разработайте иерархию геометрических фигур, которая включает в себя классы:

  • Фигура Shape. Обладает цветом и площадью. Позволяет получить тип фигуры в виде строки. При вычислении площади возвращает 0.0. При получении типа фигуры возвращает строку “Shape”.
  • Окружность Circle. Наследует свойства фигуры. Обладает радиусом, который можно изменять с помощью метода SetRadius. При получении типа фигуры возвращает строку “Circle”.
  • Прямоугольник Rectangle. Наследует свойства фигуры. Обладает шириной и высотой, которые можно изменять, используя метод SetSize. При получении типа фигуры возвращает строку “Rectangle”.

Также разработайте шаблонную функцию CalcSumArea. Она должна вычислять суммарную площадь фигур, указатели на которые находятся в переданной коллекции:

template <typename ShapeCollection>
double CalcSumArea(const ShapeCollection& collection) { ... }

Диаграмма классов фигур представлена на рисунке.

Самостоятельно решите, какие из методов базового класса следует объявить виртуальными, а какие оставить невиртуальными.

В заготовке кода есть пример использования классов и их тесты.

Ограничения

Гарантируется, что тренажёр будет передавать на вход методов и конструкторов классов Shape, Circle и Rectangle валидные значения, поэтому усложнять код этих классов проверками входных параметров не нужно.

Что отправлять на проверку

Исходный код классов, функций, перечислений и операций, которые даны в заготовке, а также необходимые для их работы директивы #include. Функция main при проверке будет заменена на функцию из тренажёра.

Как будет тестироваться ваш код

Классы Shape, Rectangle и Circle будут проверены на соответствие условиям задания.

Подсказка

  • Объявите виртуальными методы базового класса, которые должны быть переопределены в классах-наследниках.
  • Укажите спецификатор override у методов класса-наследника, которые переопределяют реализацию базового класса.
  • Площадь круга вычисляется по формуле S=πr^2. Значение константы π можно получить с использованием макроса M_PI из заголовочного файла cmath.
  • Площадь прямоугольника равна произведению ширины на высоту.
